## メモ
### データ構造
* 差分ではなくスナップショット
* 3種類のオブジェクト
 * コミットオブジェクト
 * ツリーオブジェクト
 * Blobオブジェクト
* オブジェクトは全て読み取り専用。書き換え不可。
