# index-format.txtの翻訳

Git index fileは下記の構造になっている。

全てのバイナリ番号はネットワークバイトオーダーである。
ここ記述することは、特に断りがない限りはVersion2のフォーマットについてである。

## ヘッダー

ヘッダーは下記の12バイトで構成される。

* 4バイトのシグネチャ
  シグネチャは"DIRC"であり、"dircache"の略である。
* 4バイトのバージョン番号
  サポートされているバージョンは2,3,4である。
* 32bitのインデックスエントリー数

## インデックスエントリ

 インデックスエントリのリストは、"name"昇順でソートされている。
